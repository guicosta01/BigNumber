Comece trabalhando apenas com números positivos.
• Implemente a operação de soma e teste-a à exaustão!


• Faça a subtração. Primeiramente trabalhe apenas os casos de a − b onde a > b. Em seguida
você vai ter que adicionar o suporte aos números negativos. Teste à exaustão!
• Faça testes exaustivos na soma com números negativos e positivos. Perceba que:
– a + b com a ≥ 0 e b < 0 é igual a a − |b|.
– a − b com b < 0 é igual a a + |b|.
– a + b com a < 0 e b < 0 é igual a −(|a| + |b|).
– …


• Note que utilizando essas transformações é possível utilizar o código que você já tinha feito
apenas para números positivos para tratar o caso de números negativos também!
